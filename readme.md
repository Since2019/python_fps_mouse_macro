# USAGE

## PREREQUISITE:
- pyautogui
- pynput

## START CHEAT MACHINE
- python main.py

## KEYBOARD ACTIONS:
-  Press ESC to toggle on or off
-  Press BACKSPACE to quit
-  CTRL + C won't work if you wanna quit

## MOUSE ACTIONS:
- SCROLL UP TO INCREASE SUPPRESSION
- SCROLL DOWN TO DECREASE COMPENSATION
